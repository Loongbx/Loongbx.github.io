🎮 RetroArch 使用教程：从入门到进阶

RetroArch 是一个强大的多平台模拟器前端，支持数十种游戏主机、街机平台的核心。本文将介绍如何在电视盒子、电脑或安卓设备上使用 RetroArch，畅玩经典游戏。

1️⃣ 游戏介绍及 RetroArch 安装

📌 什么是 RetroArch？

RetroArch 是一个开源模拟器前端，可以加载不同的“核心”（Core）来模拟各种老游戏机，如：

  • FC/NES、SFC/SNES、MD、GBA、PS1
  • Neo Geo、Neo Geo CD、街机（MAME、FBNeo）
  • 甚至还有 PSP、N64、Saturn（部分支持）

📥 安装方式

官网下载安装

2️⃣ 基础设置

RetroArch在安卓电视盒子运行时必须配备支持安卓设备的外设（手柄，摇杆），否则进入程序主界面会失去控制。

进入主页面如果是安卓系统盒子会显示手机界面，进入设置>驱动>菜单>ozone,让程序正常显示。

⚙️ 修改键位

修改按键映射（全局）：

设置 > 输入 >全能手柄按键绑定> 端口1控制

设置完按键映射后设置快捷键设置：

设置>快捷键>菜单开关

游戏内键位可在进入游戏后修改,独立于全局按键映射，需单独设置。

3️⃣ 文件结构说明

安装 RetroArch 后，会在设备中生成程序主文件夹RetroArch/：

目录名 功能说明
system/ 存放 BIOS 文件（非常关键）
downloads/ ROM 下载目录（可自定义）
cores/ 存放核心插件（.so/.dll 文件）
saves/ 自动保存游戏进度
states/ 快速存档/读档（状态存档）
config/ 控制器、核心、系统的配置文件

你可以在 设置 > 文件夹 中查看或修改这些路径。

4️⃣ 软件运行逻辑

RetroArch 并不是模拟器本身，而是：

  • 一个统一的前端（界面）
  • 通过加载不同 核心（Core） 来实现各种主机模拟
  • 每个核心就像“插件”，负责特定主机的运行逻辑

运行流程：

  1. RetroArch 启动主程序
  2. 用户加载某个核心（例如 FBNeo）
  3. 用户加载游戏 ROM 文件，存放ROM的文件夹可以自定义
  4. RetroArch 调用核心运行游戏

5️⃣ 核心与游戏加载指南

🔧 加载核心（模拟器插件）

  • 进入 菜单 > 加载核心 > 下载核心
  • 选择你想要的主机平台并下载核心，下载后加载核心

常用核心对照表：

主机平台 推荐核心 BIOS 需求
FC/NES Nintendo-NES/Famicom(QuickNES) 否
SFC/SNES Snes9x 否
GBA mGBA / VBA-M 否
PS1 Beetle PSX HW ✅ 需要
Neo Geo FBNeo / MAME ✅ 需要
Neo Geo CD SNK-Neo Geo CD(NeoCD) ✅ 需要
街机 FBNeo 否

NeoCD需要安装BIOS文件，下载bios文件neocd.zip,neocdz.zip,uni-bios.zip,将三个文件放入RetroArch/system/neocd/目录下即可

下载地址：[下载地址](Index of /downloads/roms/bios)

PSP需要安装BIOS文件，RetroArch已提供，菜单>在线更新>核心系统BIOS文件下载器>下载ppsspp

其它模拟器操作方法类似。

6️⃣ 游戏加载方法

🎮 单独加载步骤

  1. 打开 RetroArch
  2. Load Core 选择模拟器核心（也可跳过）
  3. Load Content 选择游戏 ROM 文件
  4. 开始游戏！

🎮 导入游戏列表

  1. 菜单>导入>手动扫描
  2. 指定文件夹
  3. 选择系统名称（非必要）
  4. 指定默认核心
  5. 开始扫描，扫描后即可在主页看到游戏列表

📦 游戏格式支持（示例）

  • FC/SFC：.nes, .smc, .sfc
  • GBA：.gba, .zip
  • PS1：.bin + .cue, .chd
  • Neo Geo CD：.chd, .cue + .bin
  • 街机：.zip（需使用对应核心支持的 ROM 集）

🧼 注意事项

  • Neo Geo CD、PS1 等游戏需要 BIOS 文件
  • 推荐使用 .chd 格式的游戏，更高效、兼容性好
  • ROM 文件不要解压（大多数核心直接识别 .zip）
